# design-pattern-nextjs

## 개요

생산성을 증대시키기 위해 Next.js를 사용한 프로젝트에서 주로 사용하는 라이브러리, 디자인 패턴에 대해 관리하고 기록합니다.


## 시작 방법

깃 레포지토리를 pull하고, 라이브러리 브랜치를 기준으로 아래의 명령어를 실행합니다.

```bash
yarn dev
```

레포지토리 별 README.md문서를 참조해 주세요.
